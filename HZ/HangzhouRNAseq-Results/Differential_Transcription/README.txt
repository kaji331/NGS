这个目录下有两个子目录。分别是两种分组情形下的alternative spling （AS） 的差异比较。
diffSplice_ControlNoRadition： 是把180cell lines按照radiation强度分成对照组（0 radiation）和辐射组。然后鉴定显著差异的AS事件。
每个子目录下有两个文件：differential_transcription.txt 和 significant.diff.AS.annotation.csv。前者是鉴定出的所有AS事件。
后者是在组间差异显著的AS事件。并且每个显著差异的AS时间都有ENSEMBNLE transcripts 和genes ID 以及gene symbols注释。
